📥 PNG / Excel / CSV 다운로드 기능

내가 담당한 기능

사용자가 예측 데이터를 다양한 형식으로 다운로드할 수 있도록 기능을 구현했습니다.

PNG, Excel, CSV 다운로드 버튼을 통해 간편하게 데이터 저장이 가능합니다.

그래프 표시 셀랙트를 통해 사용자가 보고싶은 단기, 장기, 실제 전력량 그래프를 표시하도록 구현하였습니다.

jQuery를 통해 연도 선택을 좀 더 깔끔하게 구현하였고, 지역 선택도 DB에서 바로 불러오도록 하였습니다.
