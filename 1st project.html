❄️ 냉난방 요금 납부 증명서 출력 시스템

실제 공공기관 납부 시스템을 참고하여 사용자가 자신의 냉난방비 납부 내역을 조회하고 증명서를 출력할 수 있도록 구현한 홈페이지입니다.

  • 🛠 개발 환경: JSP, MySQL DB, Java, HTML/CSS
  • 📅 개발 기간: 2025년 5월 29일~ 2025년 6월 12일 (1차 팀 프로젝트)
  • 🙋‍♂️ 기여도: DB 연동, 메인화면, 증명서 양식 디자인 및 출력 기능

🧩 주요 기능 소개

  1. 로그인 후 사용자 인증 기반으로 DB에서 요금 내역 불러오기
  2. 월별/연도별 상세 조회 가능
  3. 납부 내역을 HTML 기반 증명서 양식으로 시각화
  4. 브라우저에서 출력 및 PDF 저장 가능

📸 메인화면

📸 비로그인 고객 고객번호로 조회

기능적문제 발생

상황

고객번호를 조회 하였을경우 검색 조건을 넓게 설정 하여 관련 없는 회원도 같이 조회해버리는 문제가 발생하였습니다.

또 조회된 고객이 누적되지 않고 한명씩밖에 조회되는 문제도 있었습니다.

해결

검색 범위를 고객번호로만 바꾸고, 세션을 이용하여 고객조회 정보가 누적되도록 변경하였습니다.

📸 로그인

설명

회원 인증을 위한 로그인 페이지입니다.

사용자는 ID와 비밀번호를 입력하여 시스템에 로그인할 수 있으며, 로그인 성공 시 세션이 생성되어 안전한 접근이 가능합니다.

로그인 실패 시에는 오류 메시지가 출력되어 사용자에게 명확한 안내를 제공합니다.

📸 납부 증명서 출력

조회된 납부 내역을 증명서 형식으로 출력하거나 PDF로 저장할 수 있습니다.

📸 로그아웃

사용자 로그아웃 후 안전하게 세션이 종료됩니다.

📸 회원가입

설명

신규 사용자가 회원가입을 통해 시스템에 등록할 수 있는 페이지입니다.

회원가입 시 필수 입력 정보(ID, 비밀번호, 이름 등)를 입력하고, 가입 버튼을 통해 데이터베이스에 계정이 저장됩니다.

이메일 인증을 사용하여 사용자 본인을 확인하고 가입할 수 있게 하였습니다.

가입 완료 후에는 로그인 페이지로 이동할 수 있도록 구성했습니다.

📄 실제 증명서 출력 화면

기능 설명

조회된 납부 내역을 기반으로 증명서 HTML 양식을 출력하는 화면입니다.

사용자는 인쇄 또는 PDF 저장을 선택할 수 있습니다.

디자인 설명

공공기관 증명서 양식을 참고하여 사용자가 직관적으로 이용할 수 있도록 디자인하였습니다.

단순한 UI로 출력 시 가독성을 높였습니다.

기능적 문제 발생

상황

인쇄 화면에서 고객번호와 사용량 값을 못 가져오는 문제가 발생하였습니다.

해결

로그 역추적을 통해 컨트롤러에서 값을 못 가져오는 문제를 발견하고, DB에서 직접 값을 가져오는 방식으로 수정하였습니다.
